# Cartes postales & photos anciennes

Déposez ici les scans (format `.jpg` de préférence, ~1200 px de large).

Conseils de nommage : `acon-lieu-annee.jpg` — par exemple `acon-eglise-1910.jpg`, `acon-les-brules-1925.jpg`.

Ensuite, référencez chaque image dans `histoire.html` (voir le README à la racine).

**Droits :** ne publiez que des images libres de droits ou pour lesquelles vous avez l'autorisation.
Une carte postale ancienne n'est pas automatiquement libre de droits — vérifiez au cas par cas.
